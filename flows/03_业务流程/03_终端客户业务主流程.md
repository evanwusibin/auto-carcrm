# 终端客户业务主流程

> 流程编号：FLOW-03-03 | 版本：v1.0 | 更新时间：2026-06-12

**流程说明**：终端客户从登录到反复提问、无答案处理，到最终报修或结束会话的完整业务主流程。

---

## 完整业务主流程图

```mermaid
flowchart TD
    START([🚪 用户进入系统]) --> A{是否已登录}

    A -- 否 --> B[注册/登录<br/>手机号+密码]
    A -- 是 --> D[进入首页]
    B --> C{登录是否成功}
    C -- 否 --> C1[提示账号或密码错误]
    C1 --> B
    C -- 是 --> D

    D --> E{是否已绑定车辆}
    E -- 否 --> F[输入VIN/车牌绑定车辆]
    F --> G{车辆是否存在}
    G -- 否 --> G1[提示联系经销商激活车辆]
    G -- 是 --> H[绑定成功<br/>显示车辆信息]
    E -- 是 --> H

    H --> I[进入智能售后助手]
    I --> J[用户输入问题或描述]

    J --> K{问题是否有效}
    K -- 无效/空输入 --> K1[提示请输入具体问题]
    K1 --> K2[展示问题示例<br/>保养/质保/故障/报修]
    K2 --> J

    K -- 有效 --> L[系统识别意图]

    L --> M{问题类型路由}
    M -- 保养咨询 --> N1[查询保养手册<br/>+保养记录]
    M -- 质保咨询 --> N2[查询质保规则<br/>+车辆档案]
    M -- 故障诊断 --> N3[查询维修手册<br/>+典型案例]
    M -- 报修进度 --> N4[查询工单状态]
    M -- 其他问题 --> N5[通用RAG问答]

    N1 --> O[多路混合检索]
    N2 --> O
    N3 --> O
    N4 --> P[直接返回工单状态]
    N5 --> O

    O --> Q{检索置信度}
    Q -- 高 --> R[生成答案]
    Q -- 低 --> S[追问补充信息]
    S --> S1{用户是否补充}
    S1 -- 是 --> J
    S1 -- 否/超时 --> T[提示资料不足<br/>建议联系服务站]
    T --> U[会话继续]

    R --> V[展示答案+引用来源]
    P --> V

    V --> W{答案是否涉及故障风险}
    W -- 否 --> X[询问是否继续提问]
    W -- 是 --> Y[显示风险等级标识]

    Y --> Z{风险等级}
    Z -- 低风险 --> Z1[建议自查<br/>提供自查步骤]
    Z -- 中风险 --> Z2[建议预约服务站]
    Z -- 高风险 --> Z3[⚠️ 建议停止使用<br/>立即报修]
    Z -- 紧急 --> Z4[🆘 建议拨打救援<br/>不要启动车辆]

    Z1 --> X
    Z2 --> X
    Z3 --> AA[引导进入报修流程]
    Z4 --> AA

    AA --> BB{用户是否报修}
    BB -- 否 --> X
    BB -- 是 --> CC[生成报修单草稿<br/>自动带出车辆信息]
    CC --> DD[质保预判展示]
    DD --> EE[用户补充报修信息]
    EE --> FF{用户是否确认提交}
    FF -- 否 --> X
    FF -- 是 --> GG[✅ 提交报修单]
    GG --> HH[经销商/服务站接单]
    HH --> II[用户可查看工单进度]
    II --> X

    X --> JJ{用户是否继续提问}
    JJ -- 是 --> J
    JJ -- 否 --> KK[🔚 会话结束]

    KK --> LL{是否有完成的工单}
    LL -- 是 --> MM[引导评价服务]
    LL -- 否 --> END([结束])
    MM --> END
```

## 关键节点说明

### C-01 登录/注册
- 支持手机号+密码、短信验证码两种方式
- 首次使用需注册，注册后自动进入绑定车辆引导

### C-02 绑定车辆
- 通过 VIN（车架号）精确绑定
- VIN 不在系统中时，引导联系经销商，不能绑定不存在的车辆
- 一个账号可绑定多辆车

### C-03 发起咨询
- 输入框支持文字描述
- 可选：上传图片、输入故障码
- 系统展示问题示例引导用户描述清晰

### 风险等级建议动作

| 风险等级 | 显示样式 | 建议动作 |
|---|---|---|
| 低风险 | 🟢 绿色 | 可继续使用，参考自查步骤 |
| 中风险 | 🟡 黄色 | 尽快预约服务站检查 |
| 高风险 | 🔴 红色 | 停止使用，立即联系服务站 |
| 紧急 | 🆘 红色闪烁 | 立即停车，拨打救援热线 |
