# 终端客户业务主流程

> 流程编号：FLOW-03-03 | 版本：v1.1 | 更新时间：2026-06-13

**流程说明**：终端客户从登录、绑定车辆、提问、检索问答、故障风险判断，到报修与结束的完整主流程。

---

## 完整业务主流程图

```mermaid
flowchart TD
    A[用户进入系统] --> B{是否已登录}
    B -- 否 --> C[注册或登录]
    B -- 是 --> E[进入首页]
    C --> D{登录是否成功}
    D -- 否 --> C1[提示账号或密码错误]
    C1 --> C
    D -- 是 --> E

    E --> F{是否已绑定车辆}
    F -- 否 --> G[输入VIN或车牌绑定车辆]
    G --> H{车辆是否存在}
    H -- 否 --> H1[提示联系经销商激活车辆]
    H -- 是 --> I[绑定成功并显示车辆信息]
    F -- 是 --> I

    I --> J[进入智能售后助手]
    J --> K[用户输入问题或故障描述]
    K --> L{问题是否有效}
    L -- 否 --> L1[提示输入更具体的问题]
    L1 --> K
    L -- 是 --> M[系统识别意图]

    M --> N{问题类型}
    N -- 保养咨询 --> O1[查询保养资料]
    N -- 质保咨询 --> O2[查询质保规则]
    N -- 故障诊断 --> O3[查询维修资料和案例]
    N -- 报修进度 --> O4[查询工单状态]
    N -- 其他问题 --> O5[通用RAG问答]

    O1 --> P[六路混合检索]
    O2 --> P
    O3 --> P
    O5 --> P
    O4 --> Q[直接返回工单状态]

    P --> R{检索置信度}
    R -- 高 --> S[生成答案]
    R -- 低 --> T[追问补充信息]
    T --> T1{用户是否补充}
    T1 -- 是 --> K
    T1 -- 否 --> U[提示资料不足或建议联系服务站]

    S --> V[展示答案和引用来源]
    Q --> V

    V --> W{是否涉及故障风险}
    W -- 否 --> X[询问是否继续提问]
    W -- 是 --> Y[显示风险等级建议]

    Y --> Z{风险等级}
    Z -- 低风险 --> Z1[建议自查]
    Z -- 中风险 --> Z2[建议预约服务站]
    Z -- 高风险 --> Z3[建议停止使用并立即报修]
    Z -- 紧急 --> Z4[建议立即停车并呼叫救援]

    Z1 --> X
    Z2 --> X
    Z3 --> AA[进入报修流程]
    Z4 --> AA

    AA --> BB{用户是否报修}
    BB -- 否 --> X
    BB -- 是 --> CC[生成报修单草稿]
    CC --> DD[展示质保预判]
    DD --> EE[用户补充报修信息]
    EE --> FF{是否确认提交}
    FF -- 否 --> X
    FF -- 是 --> GG[提交报修单]
    GG --> HH[服务站接单]
    HH --> II[用户查看工单进度]
    II --> X

    X --> JJ{是否继续提问}
    JJ -- 是 --> K
    JJ -- 否 --> KK[结束会话]
```

---

## 风险等级建议动作

| 风险等级 | 显示样式 | 建议动作 |
|---|---|---|
| 低风险 | 绿色 | 可继续使用，参考自查步骤 |
| 中风险 | 黄色 | 尽快预约服务站检查 |
| 高风险 | 红色 | 停止使用，立即联系服务站 |
| 紧急 | 红色高亮 | 立即停车，拨打救援热线 |

---

## 关键节点说明

1. 登录后必须先完成车辆绑定，才能进入更准确的售后问答
2. 问题有效后进入意图识别与检索链路
3. 低置信度问题进入追问或资料不足处理
4. 高风险或紧急问题可直接引导报修
5. 用户可以在会话中多轮继续追问
