# 智能诊断流程

> 流程编号：FLOW-03-05 | 版本：v1.0 | 更新时间：2026-06-12

**流程说明**：用户描述车辆故障后，系统结合维修手册、典型案例、车辆档案进行综合诊断，输出初步诊断建议和风险等级。

---

## 完整智能诊断流程图

```mermaid
flowchart TD
    A[👤 用户选择智能诊断] --> B[选择车辆<br/>已绑定车辆列表]
    B --> C[输入故障描述]
    C --> D[可选: 上传故障图片/仪表照片]
    D --> E[可选: 输入故障码]

    E --> F[系统提取关键信息]
    F --> F1[故障现象关键词]
    F --> F2[故障码识别]
    F --> F3[部件系统判断]
    F --> F4[发生场景识别<br/>行驶中/启动时/充电时]
    F --> F5[当前里程]

    F1 --> G{信息是否充分}
    F2 --> G
    F3 --> G
    F4 --> G
    F5 --> G

    G -- 信息不足 --> H[追问补充信息]
    H --> H1[请描述具体故障现象]
    H --> H2[是否有故障码/故障灯]
    H --> H3[故障在什么情况下出现]
    H1 --> I[用户补充]
    H2 --> I
    H3 --> I
    I --> F

    G -- 信息充分 --> J[查询车辆档案]
    J --> J1[车型/VIN]
    J --> J2[购车日期/里程]
    J --> J3[保养记录]

    J1 --> K[多路混合检索]
    J2 --> K
    J3 --> K

    K --> K1[维修手册检索<br/>故障排查步骤]
    K --> K2[故障码释义检索<br/>精确匹配]
    K --> K3[典型案例检索<br/>相似故障案例]
    K --> K4[历史维修记录检索<br/>该车辆历史故障]

    K1 --> L[生成初步诊断]
    K2 --> L
    K3 --> L
    K4 --> L

    L --> M[判断风险等级]

    M --> N{风险等级}
    N -- 低风险🟢 --> O1[可继续使用<br/>提供自查步骤<br/>建议下次保养时检查]
    N -- 中风险🟡 --> O2[建议近期预约服务站<br/>提供注意事项]
    N -- 高风险🔴 --> O3[建议停止使用<br/>尽快联系服务站<br/>提供临时措施]
    N -- 紧急🆘 --> O4[立即停车<br/>不要尝试再次启动<br/>拨打救援热线]

    O1 --> P[输出诊断结果]
    O2 --> P
    O3 --> P
    O4 --> P

    P --> Q[展示引用来源<br/>维修手册章节/案例编号]

    Q --> R{用户反馈}
    R -- 问题解决/继续观察 --> S[记录诊断会话<br/>state: completed]
    R -- 需要报修 --> T[进入智能报修流程]
    R -- 还有疑问 --> U[继续多轮对话]

    U --> C
    T --> V[生成报修单<br/>带入诊断结果]
    S --> W[🔚 诊断结束]
```

---

## 风险等级判断规则

| 风险等级 | 触发条件示例 | 建议动作 |
|---|---|---|
| 🟢 低风险 | 异响但不影响驾驶、空调制冷略弱 | 自查 + 保养时检查 |
| 🟡 中风险 | 动力轻微下降、偶发故障灯 | 7天内预约服务站 |
| 🔴 高风险 | 无法启动、制动异常、电池告警 | 停止使用，立即报修 |
| 🆘 紧急 | 电池热失控、制动失灵、冒烟冒火 | 立即停车，拨打救援 |
