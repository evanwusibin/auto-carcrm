# 流程图总索引

> 文档编号：FLOW-00 | 版本：v1.1 | 更新时间：2026-06-13

本文件夹包含“商用车智能售后诊断与报修知识助手”项目的核心流程图，统一使用 Mermaid 语法编写。

为提高 Typora 兼容性，当前流程图遵循以下规范：
- 尽量避免在 Mermaid 节点中使用 emoji
- 尽量避免使用过长的 HTML 换行标签
- 优先使用简洁节点文本，详细说明写在图后正文中
- 检索链路统一以“六路召回 + RRF + 元数据过滤 + Rerank + 置信度 + Prompt 生成”为准

## 当前检索主链路（与代码一致）

1. 主体确认
2. 意图识别
3. 实体抽取
4. 六路并行召回
   - 向量检索
   - BM25 关键词检索
   - 结构化查询
   - 案例检索
   - HyDE 检索
   - Web 搜索
5. RRF 融合
6. 元数据过滤
7. Rerank 精排
8. 置信度判断
9. Prompt 构建
10. 答案生成
11. 问答持久化
