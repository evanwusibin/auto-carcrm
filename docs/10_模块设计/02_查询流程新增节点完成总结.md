## 四、核心知识点

### 4.1 节点层 vs 服务层

```
节点层（node_xxx.py）：
  - 职责：调用服务，写入 state
  - 特点：使用 @node_log 装饰器
  - 特点：使用 add_running_task / add_done_task

服务层（xxx_service.py）：
  - 职责：业务逻辑，调用 LLM
  - 特点：使用 @step_log 装饰器
  - 特点：校验参数 + 调用 LLM + 写入 state
```

### 4.2 单元测试模式

```python
