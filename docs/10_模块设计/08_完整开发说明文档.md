## 二、完整流程图

### 2.1 导入流程

```
PDF/MD 文件
    │
    ▼
node_entry（入口节点）
    │
    ├── MD 文件 → node_md_img（图片处理）
    │
    └── PDF 文件 → node_pdf_to_md（PDF转MD）
                        │
                        ▼
                    node_md_img（图片处理）
                        │
                        ▼
                    node_document_split（文档切分）
                        │
                        ▼
                    node_item_name_recognition（主体识别）
                        │
                        ▼
                    node_bge_embedding（向量化）
                        │
                        ▼
                    node_import_milvus（Milvus入库）
                        │
                        ▼
                    node_doc_meta（元数据抽取）
                        │
                        ▼
                    node_save_knowledge（知识保存）
                        │
                        ▼
                    node_publish（审核发布）
                        │
                        ▼
                       END
```

    │
    └── 无 answer → node_intent_recognition → node_entity_extraction
                                                    │
                                                    ▼
                                            ┌─────────────────────────────────────────────────────────────────────────┐
                                            │                    四路并行检索 + 可选旁路                                │
                                            │                                                                         │
                                            │  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐  │
                                            │  │  向量检索   │  │  关键词检索 │  │  结构化查询 │  │  案例检索   │  │
                                            │  │  (必选)     │  │  (必选)     │  │  (必选)     │  │  (必选)     │  │
                                            │  └──────┬──────┘  └──────┬──────┘  └──────┬──────┘  └──────┬──────┘  │
                                            │         │               │               │               │           │
                                            │  ┌─────────────┐  ┌─────────────┐                                 │
                                            │  │  [可选]     │  │  [可选]     │                                 │
                                            │  │  HyDE检索   │  │  Web搜索    │                                 │

                                                      │               │
