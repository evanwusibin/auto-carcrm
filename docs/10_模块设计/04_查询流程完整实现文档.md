```
用户问题
    │
    ▼
node_item_name_confirm（意图识别与主体确认）

    │
    └── 无 answer → node_intent_recognition（意图识别）
                        │
                        ▼
                    node_entity_extraction（实体抽取）
                        │
                        ▼
                    ┌───────────────────────────────────────────────────────────┐

                    │  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐
                    │  │  向量检索   │  │  关键词检索 │  │  结构化查询 │  │  案例检索   │

                    │         └───────┬───────┘
                    │                 │

                                      │
                                      ▼
                                node_rrf（RRF融合）
```

## 三、节点详细说明

### 3.1 node_item_name_confirm（意图识别与主体确认）

**作用**：识别用户问题中的商品名称，确认主体

**输入**：
- `original_query`：用户原始问题
- `session_id`：会话ID

**输出**：
- `item_names`：识别到的商品名称列表
- `rewritten_query`：改写后的问题
- `answer`：如果没有识别到主体，设置提示信息

**条件边**：
- 如果有 `answer` → 直接到 `node_answer_output`
- 如果没有 `answer` → 继续到 `node_intent_recognition`

---

### 3.2 node_intent_recognition（意图识别）

**作用**：分析用户问题，识别查询意图类型

**输入**：
- `rewritten_query`：改写后的问题

**输出**：
- `intent`：意图类型（6种之一）

**核心逻辑**：
```python
def recognize_intent(state):
    # 1. 校验参数
    rewritten_query = validate_intent_state(state)
    
    # 2. 获取 LLM 客户端
    llm_client = llm_provider.chat()
    
    # 3. 加载提示词
    prompt_text = load_prompt("intent_recognition", rewritten_query=rewritten_query)
    
    # 4. 调用 LLM
    messages = [HumanMessage(content=prompt_text)]
    response = llm_client.invoke(messages)
    intent_type = response.content.strip().lower()
    
    # 5. 写入 state
    state["intent"] = intent_type
    return state
```

### 3.3 node_entity_extraction（实体抽取）

**作用**：从用户问题中提取关键实体信息

**输入**：
- `rewritten_query`：改写后的问题

**输出**：
- `extracted_entities`：实体字典（JSON格式）

**核心逻辑**：
```python
def extract_entities(state):
    # 1. 校验参数
    rewritten_query = validate_entity_state(state)
    
    # 2. 获取 LLM 客户端
    llm_client = llm_provider.chat()
    
    # 3. 加载提示词
    prompt_text = load_prompt("entity_extraction", rewritten_query=rewritten_query)
    
    # 4. 调用 LLM
    messages = [HumanMessage(content=prompt_text)]
    response = llm_client.invoke(messages)
    result_text = response.content.strip()
    
    # 5. 解析 JSON
    entities = json.loads(result_text)

    state["extracted_entities"] = entities
    return state
```

```python
    state["embedding_chunks"] = normalize_retrieved_chunk(milvus_result)
```

### 3.5 node_keyword_search（关键词检索）

**作用**：使用 Milvus 过滤表达式进行精确匹配

**输入**：
- `rewritten_query`：改写后的问题
- `extracted_entities`：实体字典

**输出**：
- `keyword_chunks`：关键词检索结果

**核心逻辑**：
```python
def search_by_keywords(state):
    # 1. 提取关键词
    entities = state.get("extracted_entities", {})
    
    # 2. 构造过滤表达式
    if entities.get("fault_codes"):
        expr = f'content like "%{entities["fault_codes"]}%"'
    elif entities.get("vehicle_model"):
        expr = f'item_name like "%{entities["vehicle_model"]}%"'
    
    # 3. 调用 Milvus 查询
    results = milvus_gateway.client.query(
        collection_name=milvus_gateway.COLLECTION_NAME,
        filter=expr,
        output_fields=["chunk_id", "item_name", "title", "content", "part"],
        limit=10,
    )
    
    # 4. 格式化结果
    state["keyword_chunks"] = format_results(results)
    return state
```

- `case_chunks`：案例检索结果

**输出**：
- `rrf_chunks`：融合后的结果

**核心逻辑**：
```python
def fuse_by_rrf(state):
    # 1. 收集所有检索结果
    all_chunks = []
    all_chunks.extend(state.get("embedding_chunks", []))
    all_chunks.extend(state.get("keyword_chunks", []))
    all_chunks.extend(state.get("structured_chunks", []))
```

### 3.9 node_rerank（重排序）

**作用**：使用 Cross-Encoder 对融合结果进行重排序

**输入**：
- `rrf_chunks`：融合后的结果
- `rewritten_query`：改写后的问题

**输出**：
- `reranked_docs`：重排序后的结果

**核心逻辑**：
```python
def rerank_documents(state):
    # 1. 构造打分表
    query_list = build_question_pairs(rewritten_query, rrf_chunks)
    
    # 2. 调用 reranker 打分
    scores = reranker.compute_score(query_list)
    
    # 3. 排序
    reranked_docs = sort_by_score(rrf_chunks, scores)
    
    # 4. 写入 state
    state["reranked_docs"] = reranked_docs
    return state
```

### 3.10 node_confidence_check（置信度检查）

**作用**：检查检索结果的置信度，决定是否需要追问

**输入**：
- `reranked_docs`：重排序后的结果
- `extracted_entities`：实体字典

**输出**：
- `confidence_score`：置信度分数
- `need_followup`：是否需要追问
- `followup_question`：追问问题

**核心逻辑**：
```python
def check_confidence(state):
    # 1. 计算置信度分数
    confidence_score = calculate_confidence_score(reranked_docs)
    
    # 2. 判断是否需要追问
    need_followup, followup_question = determine_followup_need(confidence_score, entities)
    
    # 3. 写入 state
    state["confidence_score"] = confidence_score
    state["need_followup"] = need_followup
    state["followup_question"] = followup_question
    return state
```

**条件边**：
- 如果 `need_followup=True` → 设置追问问题为 answer，到 `node_answer_output`
- 如果 `need_followup=False` → 继续到 `node_answer_output`

### 3.11 node_answer_output（答案生成）

**作用**：调用 LLM 生成最终答案

**输入**：
- `reranked_docs`：重排序后的结果
- `rewritten_query`：改写后的问题
- `item_names`：商品名称列表
- `history`：历史对话记录

**输出**：
- `answer`：最终答案
- `image_urls`：图片URL列表

**核心逻辑**：
```python
def generate_answer(state):
    # 1. 检查是否已有 answer（追问问题）
    if state.get("answer"):
        return state
    
    # 2. 组装 Prompt
    prompt_text = load_prompt_text(reranked_docs, history, item_names, rewritten_query)
    
    # 3. 调用 LLM 生成答案
    llm_client = llm_provider.chat()
    response = llm_client.invoke([HumanMessage(content=prompt_text)])
    answer = response.content
    
    # 4. 提取图片 URL
    image_urls = extract_image_urls(reranked_docs)
    
    # 5. 写入 state
    state["answer"] = answer
    state["image_urls"] = image_urls
    return state
```

- `answer`：最终答案
- `image_urls`：图片URL列表
- `item_names`：商品名称列表

**输出**：无（写入 MongoDB）

**核心逻辑**：
```python
def save_qa_session(state):
    # 1. 保存用户消息
    history_repository.save_message(
        session_id=session_id,
        role="user",
        text=original_query,
        item_names=state.get("item_names", []),
    )
    
    # 2. 保存助手回答
    history_repository.save_message(
        session_id=session_id,
        role="assistant",
        text=answer,
        item_names=state.get("item_names", []),
        image_urls=state.get("image_urls", []),
    )
```
