## 二、核心代码解析

### 2.1 意图识别（真正调用 LLM）

```python
def recognize_intent(state: QueryGraphState) -> QueryGraphState:
    # 1. 校验参数
    rewritten_query = validate_intent_state(state)
    
    # 2. 获取 LLM 客户端
    llm_client = llm_provider.chat()
    
    # 3. 加载提示词
    prompt_text = load_prompt("intent_recognition", rewritten_query=rewritten_query)
    
    # 4. 构建消息
    messages = [HumanMessage(content=prompt_text)]
    
    # 5. 调用 LLM
    response = llm_client.invoke(messages)
    intent_type = response.content.strip().lower()
    
    # 6. 校验意图类型
    if intent_type not in INTENT_TYPES:
        intent_type = "after_sales_service"
    
    # 7. 写入 state
    state["intent"] = intent_type
    return state
```

**关键点**：
- 使用 `llm_provider.chat()` 获取 LLM 客户端
- 使用 `load_prompt()` 加载提示词模板
- 使用 `llm_client.invoke()` 调用 LLM

### 2.2 实体抽取（JSON 格式输出）

def extract_entities(state: QueryGraphState) -> QueryGraphState:

**关键点**：
- 提示词要求 LLM 输出 JSON 格式
- 使用 `json.loads()` 解析结果
- 只保留有效的实体类型

### 2.3 关键词检索（Milvus 过滤表达式）

```python
def search_by_keywords(state: QueryGraphState) -> QueryGraphState:
    # 提取关键词
    entities = state.get("extracted_entities", {})
    
    # 构造过滤表达式
    if entities.get("fault_codes"):
        expr = f'content like "%{entities["fault_codes"]}%"'
    elif entities.get("vehicle_model"):
        expr = f'item_name like "%{entities["vehicle_model"]}%"'
    
    # 调用 Milvus 查询
    results = milvus_gateway.client.query(
        collection_name=milvus_gateway.COLLECTION_NAME,
        filter=expr,
        output_fields=["chunk_id", "item_name", "title", "content", "part"],
        limit=10,
    )
    
    # 格式化结果
    keyword_chunks = []
    for hit in results:
        keyword_chunks.append({...})
    
    state["keyword_chunks"] = keyword_chunks
    return state
```

**关键点**：
- 使用 Milvus 的 `query()` 方法（不是 `search()`）
- 使用 `filter` 参数进行过滤
- 使用 `like` 操作符进行模糊匹配

### 2.4 案例检索（Milvus 向量检索）

```python
def search_cases(state: QueryGraphState) -> QueryGraphState:
    # 生成查询向量
    embedding_result = llm_provider.embed_documents([fault_symptom])
    dense_vector = embedding_result.get("dense")[0]
    sparse_vector = embedding_result.get("sparse")[0]

    milvus_result = milvus_gateway.hybrid_search(

        ranker_weights=(0.6, 0.4),
        norm_score=True,
        limit=5,
        output_fields=["chunk_id", "item_name", "title", "content", "part"],
    )
    
    # 格式化结果
    case_chunks = []
    if milvus_result and len(milvus_result) > 0:
        for hit in milvus_result[0]:
            entity = hit.get("entity", {})
            case_chunks.append({...})
    
    state["case_chunks"] = case_chunks
    return state
```

**关键点**：
- 使用 `llm_provider.embed_documents()` 生成向量
- 使用 `milvus_gateway.create_requests()` 构造请求
- 使用 `milvus_gateway.hybrid_search()` 混合搜索
