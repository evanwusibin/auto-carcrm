## 一、功能模块总览

系统功能分为 **6 大模块**，覆盖知识管理、智能问答、诊断报修、工单处理、案例沉淀和系统管理。

```
功能体系
├── M1：知识库管理模块       → 文档导入、解析、审核、发布、版本管理
├── M2：智能问答模块         → RAG问答、多路检索、引用溯源、反馈
├── M3：智能诊断模块         → 故障描述、风险分级、诊断建议
├── M4：智能报修模块         → 质保预判、保养判断、报修单生成
├── M5：工单处理模块         → 经销商接单、维修处理、结论填写
└── M6：系统管理模块         → 用户管理、日志分析、知识优化
```

---

## 二、第一阶段 MVP 优先级

| 优先级 | 功能模块 | 包含功能点 |
|---|---|---|
| **P0 必做** | 知识库管理 | 文档上传、解析、切分、向量化、状态管理 |
| **P0 必做** | 智能问答 | 自然语言提问、多路检索、答案生成、引用溯源 |
| **P0 必做** | 智能诊断 | 故障描述输入、意图识别、风险等级输出 |
| **P0 必做** | 智能报修 | 质保预判、保养状态查询、报修单生成 |
| **P1 应做** | 工单处理 | 经销商接单、状态流转、维修结论 |
| **P1 应做** | 用户反馈 | 点赞/点踩、错误类型标注 |
| **P1 应做** | 案例沉淀 | 维修结论转案例、审核发布 |
| **P2 可做** | 日志分析 | 高频问题、无答案分析、知识缺口 |
| **P2 可做** | 多轮对话 | 会话上下文管理、指代消解 |

## 三、各模块功能清单

### M1：知识库管理模块

| 功能编号 | 功能名称 | 优先级 | 用户角色 | 功能概述 |
|---|---|---|---|---|
| M1-01 | 文档上传 | P0 | 知识管理员、售后工程师 | 支持 PDF/Word/Excel/TXT 上传，填写元数据 |
| M1-02 | 文档解析 | P0 | 系统自动 | 按文件类型调用解析器，提取文本 |
| M1-03 | 文本清洗 | P0 | 系统自动 | 去噪、去页眉页脚、合并断行 |
| M1-04 | 文本切分 | P0 | 系统自动 | 按章节/FAQ/固定长度切分 Chunk |
| M1-05 | 向量化入库 | P0 | 系统自动 | 生成 Embedding，写入 MongoDB 向量索引 |
| M1-06 | 文档审核 | P1 | 知识管理员 | 审核解析结果，通过后发布 |
| M1-07 | 文档发布 | P1 | 知识管理员 | 发布后知识可被检索 |
| M1-08 | 文档下线 | P1 | 知识管理员 | 下线过期或废止文档 |
| M1-09 | 版本管理 | P1 | 知识管理员 | 同一文档多版本管理，新版本覆盖旧版本检索 |
| M1-10 | 导入任务查询 | P1 | 知识管理员 | 查看解析进度、失败原因 |

### M2：智能问答模块

| 功能编号 | 功能名称 | 优先级 | 用户角色 | 功能概述 |
|---|---|---|---|---|
| M2-01 | 自然语言提问 | P0 | 全部角色 | 输入自然语言问题，支持中文 |
| M2-02 | 意图识别 | P0 | 系统自动 | 识别保养/质保/故障/案例/流程等意图 |
| M2-03 | 实体抽取 | P0 | 系统自动 | 抽取车型、VIN、故障码、里程、部件 |
| M2-04 | 向量语义检索 | P0 | 系统自动 | 基于 Embedding 的语义相似度检索 |
| M2-05 | 关键词检索 | P0 | 系统自动 | BM25 关键词精确匹配 |
| M2-06 | 结构化查询 | P0 | 系统自动 | 查询车辆档案、保养记录、质保规则 |
| M2-07 | 案例检索 | P0 | 系统自动 | 检索相似故障案例 |
| M2-08 | 结果融合重排 | P0 | 系统自动 | 多路结果合并、去重、Rerank |
| M2-09 | 答案生成 | P0 | 系统自动 | 基于检索上下文调用 LLM 生成回答 |
| M2-10 | 引用溯源 | P0 | 全部角色 | 展示答案引用的文档来源和片段 |
| M2-11 | 无答案处理 | P0 | 系统自动 | 资料不足时明确提示，不编造 |
| M2-12 | 补充追问 | P0 | 系统自动 | 缺少关键信息时自动追问 |
| M2-13 | 多轮对话 | P1 | 全部角色 | 保留会话上下文，支持指代消解 |
| M2-14 | 答案反馈 | P1 | 全部角色 | 点赞/点踩/错误类型标注 |

## 四、MVP 范围确认

**第一版本交付内容（P0 功能）：**

1. ✅ 知识文档上传（PDF/Word/Excel）
2. ✅ 文档解析、清洗、切分、向量化
3. ✅ 文档状态管理（uploaded → published）
4. ✅ 自然语言问答
5. ✅ 多路混合检索（向量 + 关键词 + 结构化 + 案例）
6. ✅ 答案引用溯源
7. ✅ 无答案处理与补充追问
8. ✅ 自助故障诊断
9. ✅ 风险等级判断
10. ✅ 质保预判
11. ✅ 智能报修单生成
12. ✅ 问答日志记录
