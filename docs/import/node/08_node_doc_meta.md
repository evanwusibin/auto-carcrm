# node_08_doc_meta 元数据抽取（🆕 新增）

## 模块目的
提取文档的元数据（车型/版本/来源/有效期/标签），支持后续的过滤和权限控制。

## 流程图
```
输入：state（含 file_title, chunks, item_name）
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 1. 获取参数                                                          │
│    file_title = state.get("file_title")                             │
│    chunks = state.get("chunks", [])                                 │
│    item_name = state.get("item_name", "")                           │
└─────────────────────────────────────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 2. 调用LLM提取元数据（可选）                                          │
│    - vehicle_model（车型）                                           │
│    - version（版本号）                                               │
│    - source_type（来源：手册/FAQ/公告）                               │
│    - valid_until（有效期）                                           │
│    - tags（标签）                                                    │
└─────────────────────────────────────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 3. 写入 state                                                       │
│    state["metadata"] = {                                            │
│        "vehicle_model": "...",                                      │
│        "version": "...",                                            │
│        "source_type": "...",                                        │
│        "valid_until": "...",                                        │
│        "tags": [...]                                                │
│    }                                                                │
│    return state                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 对应service
doc_meta_service.py → extract_doc_meta(state)

## 与原项目差异
| 对比项 | 原项目 | 本项目 |
|--------|--------|--------|
| 元数据 | 只有item_name | 增加车型/版本/来源/有效期/标签 |
| LLM调用 | 无 | 可选调用LLM提取 |
| 存储位置 | 无 | 存入state["metadata"] |

## 踩坑速查
| 坑 | 现象 | 解决 |
|----|------|------|
| 正则匹配不到车型 | 车型格式不统一 | 用多种正则模式兜底 |
| LLM返回格式不对 | JSON解析失败 | 用JsonOutputParser |
| 元数据为空 | 后续过滤失效 | 给默认值兜底 |
