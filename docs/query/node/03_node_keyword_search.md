# node_keyword_search BM25关键词检索

> 文档编号：DOC-QUERY-NODE-03 | 版本：v1.0 | 更新时间：2026-06-13

---

## 一、节点目的

**用BM25算法进行关键词检索，补充向量检索可能漏掉的精确匹配。**

---

## 二、流程图

```
输入 state（含 rewritten_query, item_names）
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 1. 获取参数                                                          │
│    rewritten_query = state.get("rewritten_query")                   │
│    item_names = state.get("item_names", [])                         │
└─────────────────────────────────────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 2. 构建BM25索引                                                      │
│    从MongoDB读取所有chunks，构建BM25索引                              │
└─────────────────────────────────────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 3. BM25检索                                                          │
│    keyword_chunks = bm25_search(rewritten_query, top_k=5)           │
└─────────────────────────────────────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 4. 写入 state                                                       │
│    state["keyword_chunks"] = keyword_chunks                         │
│    return state                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

---

## 三、输入输出

| 维度 | 字段 | 类型 | 说明 |
|------|------|------|------|
| 输入 | rewritten_query | str | 改写后的问题 |
| 输入 | item_names | list[str] | 主体名称列表 |
| 输出 | keyword_chunks | list[dict] | BM25检索结果 |

---

## 四、与原项目差异

| 对比项 | 原项目 | 改造后 |
|--------|--------|--------|
| 检索方式 | 向量+HyDE+网络搜索 | 增加BM25关键词检索 |
| 用途 | 无 | 补充精确匹配 |

---

## 五、关键代码位置

- 节点：`app/process/query/agent/nodes/node_keyword_search.py`
- 服务：`app/rag/query/keyword_search_service.py`
