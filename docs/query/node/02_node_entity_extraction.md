# node_entity_extraction 实体抽取

> 文档编号：DOC-QUERY-NODE-02 | 版本：v1.0 | 更新时间：2026-06-13

---

## 一、节点目的

**从用户提问中抽取结构化实体（车型、VIN、故障码、里程、部件），用于后续过滤和查询。**

---

## 二、流程图

```
输入 state（含 original_query）
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 1. 获取参数                                                          │
│    original_query = state.get("original_query")                     │
└─────────────────────────────────────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 2. 调用LLM抽取实体                                                   │
│    entities = llm_provider.chat().invoke(                           │
│        load_prompt("entity_extraction", query=original_query)       │
│    )                                                                │
│    返回：{"vehicle_model": "...", "vin": "...", "fault_code": "..."} │
└─────────────────────────────────────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 3. 写入 state                                                       │
│    state["entities"] = entities                                     │
│    return state                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

---

## 三、输入输出

| 维度 | 字段 | 类型 | 说明 |
|------|------|------|------|
| 输入 | original_query | str | 用户原始问题 |
| 输出 | entities | dict | 抽取的实体 |

---

## 四、实体类型

| 实体类型 | 说明 | 示例 |
|----------|------|------|
| vehicle_model | 车型 | "东风天龙KL" |
| vin | 车架号 | "LGHXG6LC5NV123456" |
| fault_code | 故障码 | "P0101" |
| mileage | 里程 | "5万公里" |
| component | 部件 | "发动机" |
| time_expr | 时间表述 | "去年买的" |

---

## 五、与原项目差异

| 对比项 | 原项目 | 改造后 |
|--------|--------|--------|
| 实体抽取 | 无 | 新增，支持6类实体 |
| 用途 | 无 | 用于后续过滤和结构化查询 |
