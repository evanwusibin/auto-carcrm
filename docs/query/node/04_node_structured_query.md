# node_structured_query 结构化查询

> 文档编号：DOC-QUERY-NODE-04 | 版本：v1.0 | 更新时间：2026-06-13

---

## 一、节点目的

**查询车辆档案、保养记录、质保规则等结构化数据，为答案提供业务背景。**

---

## 二、流程图

```
输入 state（含 entities, session_id）
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 1. 获取参数                                                          │
│    entities = state.get("entities", {})                             │
│    session_id = state.get("session_id")                             │
└─────────────────────────────────────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 2. 查询车辆档案                                                      │
│    vehicle_info = vehicle_repository.find_by_vin(entities.vin)      │
└─────────────────────────────────────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 3. 查询保养记录                                                      │
│    maintenance_records = maintenance_repository.find_by_vin(...)    │
└─────────────────────────────────────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 4. 查询质保规则                                                      │
│    warranty_rules = warranty_repository.find_by_model(...)          │
└─────────────────────────────────────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 5. 写入 state                                                       │
│    state["structured_data"] = {                                     │
│        "vehicle_info": vehicle_info,                                │
│        "maintenance_records": maintenance_records,                  │
│        "warranty_rules": warranty_rules                             │
│    }                                                                │
│    return state                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

---

## 三、输入输出

| 维度 | 字段 | 类型 | 说明 |
|------|------|------|------|
| 输入 | entities | dict | 抽取的实体（含vin, vehicle_model） |
| 输出 | structured_data | dict | 结构化数据 |

---

## 四、与原项目差异

| 对比项 | 原项目 | 改造后 |
|--------|--------|--------|
| 结构化查询 | 无 | 新增，查询车辆/保养/质保 |
| 数据来源 | 只有Milvus | 增加MongoDB结构化数据 |
