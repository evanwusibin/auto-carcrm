# node_case_search 案例检索

> 文档编号：DOC-QUERY-NODE-05 | 版本：v1.0 | 更新时间：2026-06-13

---

## 一、节点目的

**检索典型案例库，找到类似故障的解决方案。**

---

## 二、流程图

```
输入 state（含 entities, rewritten_query）
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 1. 获取参数                                                          │
│    entities = state.get("entities", {})                             │
│    rewritten_query = state.get("rewritten_query")                   │
└─────────────────────────────────────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 2. 构建查询条件                                                      │
│    filter = {}                                                      │
│    if entities.get("vehicle_model"):                                │
│        filter["vehicle_model"] = entities["vehicle_model"]          │
│    if entities.get("fault_code"):                                   │
│        filter["fault_code"] = entities["fault_code"]                │
└─────────────────────────────────────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 3. 查询案例库                                                        │
│    cases = case_repository.find_by_filter(filter, limit=5)          │
└─────────────────────────────────────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 4. 写入 state                                                       │
│    state["case_chunks"] = cases                                     │
│    return state                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

---

## 三、输入输出

| 维度 | 字段 | 类型 | 说明 |
|------|------|------|------|
| 输入 | entities | dict | 抽取的实体 |
| 输入 | rewritten_query | str | 改写后的问题 |
| 输出 | case_chunks | list[dict] | 案例检索结果 |

---

## 四、与原项目差异

| 对比项 | 原项目 | 改造后 |
|--------|--------|--------|
| 案例检索 | 无 | 新增，从MongoDB案例库检索 |
| 用途 | 无 | 提供类似故障的解决方案 |
