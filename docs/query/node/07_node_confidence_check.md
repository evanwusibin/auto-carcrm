# node_confidence_check 置信度判断

> 文档编号：DOC-QUERY-NODE-07 | 版本：v1.0 | 更新时间：2026-06-13

---

## 一、节点目的

**判断检索结果的置信度，决定是否需要追问用户补充信息。**

---

## 二、流程图

```
输入 state（含 rrf_chunks, item_names）
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 1. 获取参数                                                          │
│    rrf_chunks = state.get("rrf_chunks", [])                         │
│    item_names = state.get("item_names", [])                         │
└─────────────────────────────────────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 2. 计算置信度                                                        │
│    avg_score = sum(chunk.score for chunk in rrf_chunks[:3]) / 3     │
│    has_item = len(item_names) > 0                                   │
│    confidence = avg_score * 0.8 + (0.2 if has_item else 0)          │
└─────────────────────────────────────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 3. 判断是否需要追问                                                  │
│    needs_clarify = confidence < 0.6                                 │
└─────────────────────────────────────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────────────────────────────────────┐
│ 4. 写入 state                                                       │
│    state["confidence"] = confidence                                 │
│    state["needs_clarify"] = needs_clarify                           │
│    return state                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

---

## 三、输入输出

| 维度 | 字段 | 类型 | 说明 |
|------|------|------|------|
| 输入 | rrf_chunks | list[dict] | RRF融合结果 |
| 输入 | item_names | list[str] | 主体名称列表 |
| 输出 | confidence | float | 置信度（0-1） |
| 输出 | needs_clarify | bool | 是否需要追问 |

---

## 四、与原项目差异

| 对比项 | 原项目 | 改造后 |
|--------|--------|--------|
| 置信度 | 无 | 新增，计算置信度 |
| 追问机制 | 无 | 新增，低置信度时追问 |
