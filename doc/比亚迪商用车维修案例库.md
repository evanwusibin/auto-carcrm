# 比亚迪商用车维修案例库

## 案例1：发动机故障灯亮，动力不足

**车型**：比亚迪T5  
**故障码**：P0101  
**故障现象**：发动机故障灯亮，动力不足  
**故障部件**：空气流量计  
**故障原因**：空气流量计传感器积碳导致信号异常  
**维修步骤**：
1. 使用诊断仪读取故障码
2. 检查空气流量计插头连接
3. 清洁或更换空气流量计
4. 清除故障码并试车

**维修费用**：300-800元  
**维修时间**：1-2小时  
**是否质保**：是
