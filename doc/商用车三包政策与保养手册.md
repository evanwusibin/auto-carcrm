# 商用车三包政策与保养手册

## 一、三包政策

### 1.1 法规依据
- **法规名称**：《家用汽车产品修理更换退货责任规定》（国家市场监督管理总局令第43号）
- **生效日期**：2022年1月1日
- **适用范围**：在中国境内生产、销售的家用汽车产品（含轻型商用车，总质量≤3.5吨）

### 1.2 质保期限

| 质保类型 | 标准 | 商用车常见 |
|----------|------|-----------|
| 整车质保 | 不低于3年或6万公里 | 3年/10万公里 |
| 动力总成质保 | 不低于3年或6万公里 | 5年/15万公里 |
| 三包有效期 | 不低于2年或5万公里 | 2年/5万公里 |

### 1.3 退换车条件

**退车条件**：
1. 三包有效期内，因严重安全性能故障累计进行2次修理，严重安全性能故障仍未排除
2. 三包有效期内，发动机、变速器累计更换2次后仍不能正常使用
3. 三包有效期内，发动机、变速器的同一主要零部件因其质量问题，累计更换2次后仍不能正常使用

**换车条件**：
- 三包有效期内，因产品质量问题修理时间累计超过35日
- 同一产品质量问题修理累计超过5次

- 更换冷却液

## 五、保养建议

1. 商用车载重大，保养周期应比乘用车更短
2. 柴油车必须使用符合排放标准的柴油和尿素溶液
3. 重卡新车1500公里内为磨合期，限速60km/h以内
4. 长途运输前应进行出车前检查（轮胎、制动、灯光、油水）
5. 冬季使用需更换冬季柴油（-10号/-20号/-35号）和防冻冷却液
6. 定期清理空气滤清器可显著降低油耗
7. 制动系统是商用车安全核心，建议缩短检查周期
8. DPF再生指示灯亮起时需及时进行再生操作
